# K-HELP — Korean Health Education & Literacy Program

A bilingual (Korean/English) diabetes awareness website built with **Next.js 14**, **Tailwind CSS**, and **React 18**.

## Getting Started

```bash
npm install
npm run dev
```

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start development server |
| `npm run build` | Create production build |
| `npm run start` | Serve production build |
| `npm run lint` | Run ESLint |

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **UI:** React 18, Tailwind CSS 3, Framer Motion
- **Icons:** @tabler/icons-react
- **i18n:** Custom `LanguageContext` with Korean/English translations
- **Fonts:** Noto Sans KR, Source Sans 3
